📋 How to use VisionFlow:

  1. Set the SOURCE path with files to process
  2. Drag blocks from the left to build your pipeline
  3. Configure each block's properties
  4. Download your pipeline as .vf file
  5. Run: python main.py your_pipeline.vf
  6. Check processed_output/ for results
Glob patterns: Use wildcards to match files
  • "videos/*.mp4" - All .mp4 in videos folder
  • "photos/**/*.jpg" - All .jpg recursively
  • "image.jpg" - Single file
